Colore l'écran selon la sortie du réseau de neurones.

Chaque pixel est évalué par le réseau pour estimer sa classe, puis colorié en conséquence.

Colore l'écran en RGB pour les réseaux multi-classes.

Utilisé lorsqu’il y a plus de deux classes. Chaque classe est associée à une couleur distincte. Le rendu final utilise un mélange RGB basé sur la sortie du réseau pour chaque pixel.

void generer_spirales_archimede ( )

Génère deux spirales d'Archimède entrelacées pour la classification binaire.

Génère deux spirales opposées (bleu et rouge), normalisées dans [-1, 1].

Génère deux spirales d'Archimède pour un problème de classification binaire.
