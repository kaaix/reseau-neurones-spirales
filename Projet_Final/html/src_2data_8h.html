Detailed Description

Fonctions et structures pour la génération de données (spirales) et la visualisation.

Ce module gère :

  • La génération de jeux de données en forme de spirales pour l'entraînement des réseaux de neurones
  • La génération personnalisée pour un nombre variable de classes
  • La coloration de l'écran selon les sorties du réseau
  • Le stockage des points dessinés manuellement

Colore l'écran en RGB pour les réseaux multi-classes.

Utilisé lorsqu’il y a plus de deux classes. Chaque classe est associée à une couleur distincte. Le rendu final utilise un mélange RGB basé sur la sortie du réseau pour chaque pixel.

◆ generer_spirales_personnalisees()

void generer_spirales_personnalisees ( int nb_classes)

Génère un ensemble de spirales pour un nombre arbitraire de classes.

Chaque spirale est décalée angulairement pour représenter une classe différente. Les points sont normalisés dans [-1, 1] et stockés dans dataset.

Parameters
nb_classesNombre de classes (spirales) à générer.

Génère un ensemble de spirales pour un nombre arbitraire de classes.

Génère des spirales personnalisées pour plusieurs classes.

Les points sont générés et stockés dans le tableau global dataset_P.

Parameters
nb_classesNombre de classes à générer (maximum : MAX_CLASSES).
